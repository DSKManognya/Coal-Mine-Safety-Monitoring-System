# Coal Mine Safety Monitoring System

An IoT-based real-time coal mine safety monitoring system developed using Arduino Uno, LoRa communication, and environmental sensors to detect hazardous conditions in underground mines.

## Features

- Real-time environmental monitoring
- Wireless LoRa communication
- Gas, temperature, vibration and motion sensing
- Threshold-based hazard detection
- Buzzer alert system

## Technologies Used

- Arduino Uno
- LoRa
- MQ-2 Gas Sensor
- MPU6050
- LM393
- Arduino IDE
